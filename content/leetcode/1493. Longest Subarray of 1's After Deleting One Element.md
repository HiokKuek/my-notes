Given a binary array `nums`, you should delete one element from it.

Return _the size of the longest non-empty subarray containing only_ `1`_'s in the resulting array_. Return `0` if there is no such subarray.

E.g. 
**Input:** nums = [1,1,0,1]
**Output:** 3
**Explanation:** After deleting the number in position 2, [1,1,1] contains 3 numbers with value of 1's.

### My Solution 
#### Intuition 
We keep track of the number of 1s on the left as well as on the right of the zero. Whenever you see a zero, we will reset the the count, since what's on the left will become what's on the right. 
#### Implementation 
```python 
class Solution:
	def longestSubarray(self, nums: List[int]) -> int:
	zero_l, zero_r = 0, 0
	zeroed = False
	res = 0
	
	for num in nums:
		if zeroed:
			if num == 0:
				total = zero_l + zero_r
			if total > res:
				res = total
			zero_l = zero_r
			zero_r = 0
		else:
			zero_r += 1
	else:
		if num == 0:
			zeroed = True
		else:
			zero_l += 1
	
	total = zero_l + zero_r
	
	if total > res:
		res = total
	
	return res if zeroed else res - 1
```
